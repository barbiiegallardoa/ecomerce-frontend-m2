# Espacio Natural - E-commerce

## Descripción
Proyecto desarrollado como parte del Módulo 2 - E-commerce Frontend.
Se trata de una tienda en línea ficticia llamada "Espacio Natural", especializada en productos artesanales y naturales.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Funcionalidades
- Visualización de productos en grilla responsiva.
- Página de detalle de cada producto.
- Carrito de compras simulado usando localStorage.
- Contador de items en el navbar.
- Eliminar productos y vaciar carrito.

## Estructura del proyecto
- `index.html`: Página principal con listado de productos.
- `producto.html`: Página de detalle del producto.
- `carrito.html`: Página para ver y gestionar el carrito.
- `img/`: Carpeta con las imágenes de los productos.
